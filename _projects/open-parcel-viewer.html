--- name: Open Parcel Viewer img: parcel-wide-ii.jpg caption: Open Source County Parcel Viewer using Leaflet about: >- A static html, javascript and jquery parcel web map, capable of rendering up to 50k polygons on desktop and mobile. The app uses Leaflet as the mapping api and Jekyll to build the static pages. Color, initial map center/zoom and the search field are defined in the yaml front matter, with minimal to zero extra coding needed.
